We use data from 
- Kingston, RI to represent the MIDATL region
- Elkins, WV to represent the Western region
- Charlottesville, VA to represent the SOUTH region


Data is obtained from : https://www.ncei.noaa.gov/pub/data/uscrn/products/hourly02/
Data README: https://www.ncei.noaa.gov/pub/data/uscrn/products/hourly02/readme.txt
Data FTP access: ftp://ftp.ncdc.noaa.gov/pub/data/uscrn/products/hourly02


RAW FILES IN L2 (and possibly L4) are not included in the repo. They are to0 large.
